# YUMO PULSE

A compact DIY gadget built on the ESP32-C3 Super Mini.
Displays live weather data, clock, and proximity detection
on a tiny 0.09" OLED — with a web interface you can control from your phone.

Built by [YUMO BUILDS](https://www.youtube.com/@yumobuilds)

---

## 🔧 Hardware

- ESP32-C3 Super Mini
- SHT3X Temperature & Humidity Sensor
- 0.09" OLED Display (128×32)
- Multiple LEDs (seconds blink, glow, flash, proximity)
- Bare-wire touch/proximity antenna on GPIO7

---

## 🚀 Features

- 🕐 **Clock** — London time via NTP with day & date
- 💧 **Temperature & Humidity** — live SHT3X readings with pixel icons
- ✨ **Brand page** — sine-wave contrast pulsing synced to LEDs
- 🌐 **Web interface** — live sensor data + send text to OLED from browser
- 🤚 **Proximity detection** — self-calibrating, LED glows as hand approaches
- 🌙 **Night mode** — display and LEDs dim automatically midnight–5am
- 📶 **WiFiManager** — captive portal on first boot, saves credentials
